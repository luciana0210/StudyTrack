# StudyTrack - Sistema de Planejamento de Estudos
# Sobre o projeto 
StudyTrack é um sistema web completo para planejamento e acompanhamento de estudos. Ele permite cadastrar disciplinas, definir metas, registrar horas estudadas e visualizar progresso em tempo real através de um dashboard interativo.
# Funcionalidades implementadas
- Cadastro de disciplinas
- Nome e carga horária prevista
- Controle total de horas planejadas vs realizadas
- Gerenciamento de metas
- Associação com disciplinas específicas
- Acompanhamento de progresso por matéria
- Registro de horas estudadas
- Cálculo automático de horas totais
- Dashboard inteligente
# Tecnologias Utilizadas 
Frontend: HTML5 | CSS3 | Vanilla JavaScript
Armazenamento: LocalStorage (PWA Ready)
Design: Mobile-First | Gradientes | Animações CSS
Responsivo: 100% adaptável (Desktop/Mobile/Tablet)
